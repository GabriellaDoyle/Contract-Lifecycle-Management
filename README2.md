** Contract Lifecycle Management (AI-Powered Demo on SAP BTP)
Overview
This project is an AI-powered Contract Lifecycle Management (CLM) application developed as a demonstration within SAP Innovation Labs. The purpose of the demo is to showcase how artificial intelligence can support business processes by automating the generation of legal-style documents.
The application allows users to input prompts and generate structured contract outputs, demonstrating how enterprise technologies can integrate AI to improve efficiency, consistency, and scalability in document workflows.
The project follows a full development lifecycle, including local development, version control via GitHub, and deployment to SAP Business Technology Platform (BTP) Cloud Foundry.

** Key Features

AI-generated legal-style document creation
Prompt-based input system
Cloud deployment on SAP BTP (Cloud Foundry)
Secure handling of API keys using environment variables
Demo-ready interface for business and customer presentations
Technologies Used
JavaScript / HTML / CSS
AI API integration
Git & GitHub (Version Control)
SAP Business Technology Platform (BTP)
Cloud Foundry CLI

** Architecture Overview

The application follows a simple architecture:
Frontend: User interface for input and displaying generated contracts
Backend Logic: Handles AI API requests and response processing
Cloud Deployment: Hosted on SAP BTP Cloud Foundry
External Dependency: AI service API (secured via environment variables)
